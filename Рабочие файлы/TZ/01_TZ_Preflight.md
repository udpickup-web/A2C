# 01_TZ_Preflight — Preflight (рамка, стандарт, единицы, масштаб)
Версия: 1.0 • Дата: 2025-10-02

## Цель
- Определить рабочую область, штамп; определить стандарт (ГОСТ/ISO/ASME), единицы (мм/дюймы), масштаб.

## Вход
- Изображение/страница PDF; параметры порогов.

## Выход
- `preflight.json` по схеме; debug‑картинки.

## Алгоритм/требования к реализации
- OpenCV: морфология, детект линий/рамки; эвристики по штампу/пиктограммам; default `mm` при отсутствии явной метки.

## API/Интерфейсы
- `POST /preflight` и CLI util.

## Критерии приёмки
- JSON валиден, поля заполнены; время ≤ 500 мс на A4@300DPI.

## Артефакты
- `preflight.json`, `*_frame_debug.png`.
